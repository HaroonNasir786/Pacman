# Pacman
 C++ Game 
 
 ![Image of pacman]( https://github.com/HaroonNasir786/Pacman/blob/main/4.png)
 
## How to play the game:
-	One lifeline (pacman)
###	Controls:
- W – GO UP
- S – GO DOWN
- D – GO RIGHT
- A - GO LEFT

### Additional controls:
- P- Pause/Resume the game
- C- Randomly generate Cherries
- Space bar – start the game 
- Shift + D – boost speed

## Each munchie / cherry worth 1 point
